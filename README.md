# Workforce-Analytics-Dashboard.

This project presents an interactive Workforce Analytics Dashboard built using Microsoft Power BI to help organizations understand, monitor, and analyze key human resources metrics.

The dashboard provides clear insights into employee demographics, workforce distribution, attrition patterns, and overall workforce performance, enabling data-driven decision-making for HR teams and management.
